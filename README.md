# 🖥️ IT Envanter Takip

Modern ve kullanıcı dostu bir IT envanter yönetim sistemi. Bilgisayar, yazıcı, monitör, ağ ekipmanları ve diğer IT varlıklarını tek bir panelden kolayca takip edin.

## 🌐 Demo

👉 **[Canlı Demo](https://Alpsencers.github.io/it-envanter/)**

## ✨ Özellikler

- **Dashboard** — Toplam cihaz, aktif, bakımda, arızalı ve toplam değer istatistikleri
- **Grafikler** — Kategori dağılımı (doughnut) ve durum bazlı (bar) Chart.js grafikleri
- **Uyarı Sistemi** — Garanti süresi dolan ve bakımı geciken cihazlar için otomatik uyarılar
- **CRUD İşlemleri** — Cihaz ekleme, düzenleme ve silme
- **Arama & Filtreleme** — Cihaz adı, marka, model, seri no ve atanan kişiye göre arama; tür, durum ve sıralama filtreleri
- **Karanlık / Aydınlık Tema** — Tek tıkla tema geçişi, tercih localStorage'da saklanır
- **Veri Dışa / İçe Aktarma** — JSON formatında yedekleme ve geri yükleme
- **Responsive Tasarım** — Masaüstü ve mobil cihazlarda uyumlu görünüm
- **localStorage** — Tüm veriler tarayıcıda saklanır, sunucu gerektirmez


## 🛠️ Kullanılan Teknolojiler

- **HTML5** — Tek sayfa uygulama yapısı
- **CSS3** — CSS Custom Properties, Grid, Flexbox, animasyonlar
- **Vanilla JavaScript** — Framework bağımlılığı yok
- **Chart.js 4.4** — İnteraktif grafikler
- **Google Fonts** — DM Sans, Outfit, JetBrains Mono

## 🚀 Kurulum

Projeyi yerel ortamınızda çalıştırmak için:

```bash
git clone https://github.com/Alpsencers/it-envanter.git
cd it-envanter
```

Ardından `index.html` dosyasını tarayıcınızda açın. Herhangi bir build adımı veya sunucu gerekmez.
